# MetaData
This program shows all the metadata of an image that is in your pc and you can also strip the metadata also this is my first time making something on git so please show some support.

lightweight Python tool to extract, analyze, and purge tracking metadata (EXIF & Text Chunks) from images.

##  Features
-  **Multi-Format Processing**: Extracts data seamlessly from JPEGs, PNGs, and WebPs.
-  **Google Earth Alignment**: Converts complex GPS metrics into direct copy-pasteable coordinates.
-  **Zero-Loss Scrubbing**: Strips structural metadata without re-compressing or degrading pixel quality.

##  Installation
Make sure you have Python installed, then install the Pillow and  Tkinter libraries

## 💻 Usage
Simply run the script and select your image using the file browser popup.
